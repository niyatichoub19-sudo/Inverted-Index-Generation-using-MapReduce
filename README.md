# Inverted-Index-Generation-using-MapReduce
This project implements a distributed text processing system that builds an inverted index from multiple documents stored in HDFS using Java-based Mapper, Reducer, and Driver classes.
# Inverted Index Generation using MapReduce

## 📌 Overview
This project implements an **Inverted Index** using the Hadoop MapReduce framework. An inverted index maps each word to the list of documents in which it appears, enabling efficient search and retrieval in large-scale distributed systems.

---

## 🎯 Objective
- Understand distributed data processing using Hadoop MapReduce
- Perform text preprocessing (tokenization, normalization)
- Implement key-value based computation
- Generate scalable inverted index from multiple documents

---

## 🏗️ Technologies Used
- Apache Hadoop
- Java
- HDFS (Hadoop Distributed File System)
- MapReduce Programming Model
